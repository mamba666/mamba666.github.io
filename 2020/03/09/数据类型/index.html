数据类型

发表于 2020-03-09

数据类型远不止基础学到的那些东西

分类

整体上分为两大类：基本（值）类型；对象（引用）类型。

基本（值）类型

  • String
  • Boolean
  • Number
  • Null
  • Undefined

对象（引用）类型

> Object   以下都是Object
> Function  可以执行的对象
> Array     有下标的对象，并且数据都是有顺序的

判断

  • typeof 返回数据类型的字符串表达式
  • instanceof 专门判断对象的具体类型（函数还是数组）
  • === 全等，不会做类型的转换，不要使用==。

A instanceof B : 指A是否为B的一个实例化对象，正确返回true，错误返回false。

这里要特别注意，在使用对象的属性时，一定不要直接写属性，一定一定要在属性前带上对象名。
