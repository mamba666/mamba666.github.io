前期准备工作

首先要设计好各个文件夹的分工。

  1. 图片类：图片可以分为两类，一类是样式类的，比如logo、background等；还有一类，比如电商网站的轮播图、商品图，因为这些都是需要频繁从后台更换的。所以可以分为img和upload。
  2. 样式类：样式可以分为两类，一类是公共样式，比如每个页面都包含的header、footer、还有版心；另外一类是初始化样式，确保各个浏览器显示同一。所以可以分为common.css和base.css
  3. 其他的就是一些iconfont、js。没啥可说的。

网站的优化三大标签

SEO是由英文Search Engine Optimization缩写而来， 中文意译为“搜索引擎优化”！

SEO是指通过对网站进行站内优化、网站结构调整、网站内容建设、网站代码优化等和站外优化，从而提高网站的关键词排名以及公司产品的曝光度。 简单的说就是，把产品做好，搜索引擎就会介绍客户来。

以下代码举例均为JD例子

title

title具有不可替代性，是我们的内页第一个重要标签，是搜索引擎了解网页的入口，和对网页主题归属的最佳判断点。

description

对于关键词的作用明显降低，但由于很多搜索引擎，仍然大量采用网页的MATA标签中描述部分作为搜索结果的“内容摘要”。 就是简要说明我们网站的主要做什么的。

我们提倡，Description作为网站的总体业务和主题概括，多采用“我们是…”“我们提供…”“×××网作为…”“电话：010…”之类语句。

keywords

Keywords是页面关键词，是搜索引擎关注点之一。Keywords应该限制在6～8个关键词左右，电商类网站可以多 少许。

快捷导航栏 shortcut 左侧部分 实现技术的选择

两家实现的技术选择各不相同

TB：在外层使用p包裹，欢迎语句使用em，注册登录使用a。这里使用a的原因是注册登录需要跳转到别的页面。

JD：直接整个shortcut都是用了ul包裹，左右两部分均为li。

这里不去谈论谁好谁坏，因为只去制作左侧部分，所以选用TB方案。

快捷导航栏 shortcut 右侧部分

两家的右侧部分都是使用的ul包裹li实现的。

各自的做法：

JD：每一个li之间都是用了竖直的小线去分隔开，然后在li里面包裹a（a里面是显示的文字），并且有的li在鼠标移上去时会有下拉菜单显示。

TB：并没有在每个li之间都使用竖直的小线分开，而是通过相关性进行的分割。并且，它和JD不同之处在于，它是li包裹div，div在包裹a，即每个li里面比JD都要多一个div。

没有谁好谁坏，这里使用JD方案。

快捷导航栏 shortcut 总结

  1. 当有多个颜色需要重复使用的时候，可以单独写一个类去设置，之后再用的时候，直接调用这个类就可以。
  2. li之间的竖直的小线可以使用li做出来，即每一个竖直的小线是都是一个li。
  1. 在使用ul li时，会垂直显示，那么如何显示成水平的呢？让每一个li都左浮动就行
  1. 在使用iconfont的下箭头时，文字会默认对齐他的底部。所以当icon图标过大，文字就会下移。但是icon实际上还是垂直居中于这个li。多用f12找问题。

先分析一下现在网站的头部怎么写的

可以发现，基本上都是分为左中右三个模块，分别为logo、search、二维码或者购物车等等。

以JD举例子，它是最外层一个div包裹一个div（版心），然后在包裹了logo、search等

头部 header 右侧 logo

logo并不只是放一张图片这么简单，其中有很多学问。

先看看JD的logo的html结构：

  1. 可以发现里面是用了h1标签，h1又包裹了a。
  2. 并且div h1 a三个标签都是一样大的
  3. 由于a标签是行内，所以要使用display:block
  • logo 里面 首先放一个 h1 标签 ，目的是为了提权，告诉搜索引擎，这个地方很重要
  • h1里面在放一个链接，可以返回首页的， 给链接一个 大小 和 logo 的背景图片
  • 连接 里面要放文字（网站名称），为了搜索引擎收录我们。 但是文字不要显示出来
    • 要用 text-indent 移到盒子外面 （text-indent: -9999px) 然后overflow:hidden 淘宝的做法
    • 直接给font-size: 0; 就看不到文字了， 京东的做法。
  • 最后给 连接一个 title 这样鼠标放到logo 上， 就可以看到提示文字了
